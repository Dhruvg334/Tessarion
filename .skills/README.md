# Tessarion Skills

This folder is reserved for Tessarion-specific skills, workflows, reusable instructions, evaluation rubrics, and task-specific guidance.

## Purpose

Skills in this directory can be used by AI coding agents to understand Tessarion-specific patterns, conventions, and workflows.

## Structure

- Place public-safe skills directly in this folder
- Private or draft skills should go in `.skills/private/` or `.skills/drafts/` (these are gitignored)
- Files ending in `.local.md` are gitignored

## Guidelines

- Do not place API keys, secrets, or sensitive information in skill files
- Do not place personal instructions or private planning notes here
- Skills should be reusable and well-documented
